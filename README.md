# 悦成小阁民宿落地页

这是“悦成小阁”民宿宣传落地页。页面是纯静态项目，可以直接双击 `index.html` 打开，也可以部署到 GitHub Pages。

## 打开方式

1. 打开本文件夹：`D:\AI space\yuecheng-landing`
2. 双击 `index.html`
3. 浏览器中应看到“悦成小阁”首屏、入住指南交互模块、古城路线和联系方式

## 素材说明

- `assets/photos/hero-real-architecture.png`：首屏门面照片
- `assets/photos/rooms/facade-beautified-close.png`、`facade-beautified-street.png`：门面段使用的美化后实拍
- `assets/photos/rooms/`：房间、浴室、书桌、投影、茶具和门面实拍素材
- `assets/photos/lock-bell.png`、`parking.png`、`power-card.png`：入住说明配图
- `assets/generated/route-illustration.png`：古城路线氛围底图，不作为真实地图

## GitHub Pages

发布时只需要包含根目录静态文件和 `assets/`。`tmp/` 是本地检查与截图目录，不需要发布。
